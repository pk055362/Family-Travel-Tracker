# Family Travel Tracker 🌍

A full-stack web application that helps families track and visualize the countries they have visited around the world. Each family member can be assigned a unique color, making it easy to identify their travel history on an interactive world map.

Built using **Node.js, Express.js, PostgreSQL, EJS, HTML, CSS, and JavaScript**.

---

## ✨ Features

- 👨‍👩‍👧‍👦 Add multiple family members
- 🌎 Track visited countries for each member
- 🎨 Unique color representation for every family member
- 🗺️ Interactive world map visualization
- ➕ Add new visited countries dynamically
- 📊 Store and manage travel data using PostgreSQL
- ⚡ Responsive and user-friendly interface

## 🚀 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/pk055362/Family-Travel-Tracker.git
```

### 2️⃣ Navigate to the Project Directory

```bash
cd Family-Travel-Tracker
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Setup PostgreSQL Database

Create a PostgreSQL database and import the required tables using:

```sql
CREATE DATABASE family_travel_tracker;
```

Run the SQL queries provided in the project.

Update your PostgreSQL credentials inside the project configuration.

---

## ▶️ Run the Application

```bash
node index.js
```

or (if nodemon is installed)

```bash
nodemon index.js
```

The app will run on:

```bash
http://localhost:3000
```
